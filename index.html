WHAT IS PULSE?

Pulse is A minimal CSS framework, designed to prevent you from having to type the same lines of code everytime you start a new project. We focused in on the simple things such as grids, buttons, lists and typography.

Pulse is 100% responsive meaning itâ€™s mobile friendly and your able to resize your browser as much as you like. Pulse Is Open Souce and licensced under the MIT license You Can View The Github page here. Pulse is ~3kb when minified which means it loads super fast.

DOCMENTATION

Getting Started

Install With NPM

Run npm install pulsecss then link to ./node_modules/pulse/dist/pulse.css


Install With Bower

Run bower install pulse then link to ./bower_components/pulse/dist/pulse.css


Link to our site

Link to us via <link type="text/css" rel="stylesheet" src="https://getpulse.ml/dist/pulse.min.css"/>

Buttons

Pulse has 2 variations of buttons with more coming soon, but for now you have the option of filled or outlined buttons.


Standard (Filled) Button

This is the default style for buttons

<button>Test</button>

or add

<input class="btn">Test</input>

Standard (Filled) Button

This is the default style for buttons

<button class="btn-outline">Test</button>

or add

<input class="btn-outline">Test</input>

Lists

Pulse offers two types of lists, ordered and unordered

Unordered List

  • Item 1
  • Item 2
  • Item 3
			<ul>
   <li>Item 1</li>
   <li>Item 2</li>
   <li>Item 3</li>
</ul>

Ordered List

  1. Item 1
  2. Item 2
  3. Item 3

   <li>Item 1</li>
   <li>Item 2</li>
   <li>Item 3</li>
</ol>

Typography

Pulse is built on a responsive typography system based on em's.

The default font for headers is Montserrat, Montserrat is a free font served to pulse via Google Fonts, which can be easily changed if you required a different font
